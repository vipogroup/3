## 🎨 פלטת צבעים ראשית

### צבעים עיקריים (Primary Colors)
```css
--primary-color: #3498db        /* כחול ראשי - לא בשימוש נרחב */
--secondary-color: #2980b9      /* כחול כהה - לא בשימוש נרחב */
--accent-color: #5dade2         /* כחול בהיר (אקסנט) */
```

### צבעים בשימוש אקטיבי (הצבעים האמיתיים של האתר)
```css
צבע כחול כהה עיקרי: #1e3a8a     /* כחול נייבי - צבע דומיננטי */
צבע טורקיז/ציאן: #0891b2         /* טורקיז בהיר - צבע משני */
צבע טורקיז בהיר: #06b6d4         /* טורקיז בהיר יותר - למצבי hover */
```

### צבעי רקע וטקסט
```css
--light-color: #f8f9fa          /* רקע בהיר - לבן מעט אפרפר */
--dark-color: #2c3e50           /* טקסט כהה */
--text-color: #2c3e50           /* צבע טקסט ראשי */
--text-light: #ffffff           /* טקסט בהיר - לבן */
--gray-color: #7f8c8d           /* אפור */
```

### צבעי משנה
```css
צבע זהב: #d4af37                /* זהב - לאייקונים חברתיים בהובר */
צבע כתום: #fbbf24, #f59e0b      /* כתום - לתגיות "הכי משתלם" */
צבע כוכבים: #f59e0b             /* כתום - לדירוג כוכבים */
צבעי אפור: #4b5563, #6b7280, #374151  /* גווני אפור לטקסטים משניים */
```

## 🎯 גרדיאנטים (Gradients)

### גרדיאנט ראשי (הכי נפוץ)
```css
linear-gradient(135deg, #1e3a8a 0%, #0891b2 100%)
```
**שימוש:** כפתורים, אייקונים, מסגרות, רקעים

### גרדיאנט משני
```css
linear-gradient(135deg, #0891b2 0%, #1e3a8a 100%)
```
**שימוש:** כפתורים בהובר (הפוך)

### גרדיאנט אנכי
```css
linear-gradient(180deg, #1e3a8a 0%, #0891b2 100%)
```
**שימוש:** קווים אנכיים (timeline)

### גרדיאנט רקע Hero
```css
linear-gradient(
    to bottom,
    rgba(13, 60, 97, 0.85) 0%,
    rgba(13, 60, 97, 0.55) 100%
)
```
**שימוש:** אוברליי על תמונת רקע ראשית

## 🎨 סגנון עיצובי כללי

### עקרונות עיצוב
1. **מודרני ונקי** - עיצוב מינימליסטי עם הרבה רווח לבן
2. **גרדיאנטים** - שימוש נרחב בגרדיאנטים כחול-טורקיז
3. **מסגרות גרדיאנט** - border gradient בעזרת טכניקת background-clip
4. **צללים עדינים** - box-shadow עם שקיפות נמוכה
5. **אנימציות חלקות** - transitions של 0.3s ease
6. **עיצוב תלת-ממדי** - שימוש ב-transform, perspective, translateZ

### פינות מעוגלות (Border Radius)
```css
--border-radius: 8px              /* ברירת מחדל */
border-radius: 10px-16px          /* כרטיסים */
border-radius: 50px               /* כפתורים עגולים */
border-radius: 50%                /* אייקונים עגולים */
```

### צללים (Box Shadows)
```css
/* צל עדין */
box-shadow: 0 4px 15px rgba(52, 152, 219, 0.1);

/* צל בינוני */
box-shadow: 0 4px 20px rgba(8, 145, 178, 0.1);

/* צל חזק */
box-shadow: 0 8px 30px rgba(8, 145, 178, 0.3);

/* צל בהובר */
box-shadow: 0 10px 30px rgba(8, 145, 178, 0.25);
```

## 🔄 אלמנטים חוזרים

### 1. מסגרת גרדיאנט (Gradient Border)
**טכניקה:** שימוש ב-background-image עם שתי שכבות
```css
border: 2px solid transparent;
background-image: linear-gradient(white, white), 
                  linear-gradient(135deg, #1e3a8a, #0891b2);
background-origin: border-box;
background-clip: padding-box, border-box;
```
**שימוש:** כרטיסי מוצרים, כרטיסי FAQ, טפסים, קופסאות מידע

### 2. כפתורים (Buttons)

#### כפתור ראשי
```css
background: linear-gradient(135deg, #1e3a8a 0%, #0891b2 100%);
color: white;
padding: 8px 16px;
border-radius: 8px (או 50px לכפתורים עגולים);
```

#### כפתור משני
```css
background: transparent;
color: white;
border: 2px solid white;
```

#### אפקט Hover
```css
transform: translateY(-2px) או translateY(-5px);
box-shadow: 0 6px 25px rgba(8, 145, 178, 0.3);
```

### 3. אייקונים עגולים
```css
width: 50px-76px;
height: 50px-76px;
background: linear-gradient(135deg, #1e3a8a 0%, #0891b2 100%);
border-radius: 50%;
color: white;
box-shadow: 0 4px 15px rgba(8, 145, 178, 0.2);
```

### 4. כותרות סקציות
```css
color: #1e3a8a;
font-size: 1.6rem;
font-weight: 700;
text-align: center;
```

**קו תחתון מונפש:**
```css
::after {
    width: 80px;
    height: 4px;
    background: linear-gradient(90deg, #1e3a8a 0%, #0891b2 50%, #1e3a8a 100%);
    animation: gradientShine 3s ease-in-out infinite;
}
```

## ✨ אנימציות ואפקטים

### אנימציות מוגדרות

#### 1. floatPulse
```css
@keyframes floatPulse {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-6px); }
}
```
**משך:** 3.5s
**שימוש:** אייקונים מרחפים

#### 2. staggerFadeIn
```css
@keyframes staggerFadeIn {
    from { opacity: 0; transform: translateY(20px); }
    to { opacity: 1; transform: translateY(0); }
}
```
**משך:** 0.6s
**שימוש:** כותרות, טקסטים

#### 3. gradientShine
```css
@keyframes gradientShine {
    0% { background-position: -200% center; }
    100% { background-position: 200% center; }
}
```
**משך:** 3s
**שימוש:** קו תחתון של כותרות

#### 4. pulse
```css
@keyframes pulse {
    0% { box-shadow: 0 0 0 0 rgba(52, 152, 219, 0.7); }
    70% { box-shadow: 0 0 0 10px rgba(52, 152, 219, 0); }
    100% { box-shadow: 0 0 0 0 rgba(52, 152, 219, 0); }
}
```
**משך:** 2s infinite
**שימוש:** כפתור CTA ראשי

#### 5. bounce
```css
@keyframes bounce {
    0%, 20%, 50%, 80%, 100% { transform: translateY(0); }
    40% { transform: translateY(-10px); }
    60% { transform: translateY(-5px); }
}
```
**משך:** 2s infinite
**שימוש:** חץ גלילה

### אפקטי Hover נפוצים
```css
/* הרמה */
transform: translateY(-3px) או translateY(-8px);

/* הגדלה */
transform: scale(1.1) או scale(1.15);

/* סיבוב תלת-ממדי */
transform: translateZ(60px) scale(1.12) rotateY(5deg);

/* שינוי צבע רקע */
background: rgba(255, 255, 255, 0.25);
```

## 📱 עיצוב לפי סקציות

### 1. Navbar (תפריט ניווט)
- **רקע:** שקוף לגמרי (transparent)
- **טקסט:** לבן (rgba(255, 255, 255, 0.9))
- **גובה:** 70px
- **אייקונים:** גרדיאנט עם stroke לבן
- **Hover:** רקע rgba(255, 255, 255, 0.15)

### 2. Hero Section (סקציה ראשית)
- **רקע:** תמונה עם אוברליי גרדיאנט כחול כהה
- **טקסט:** לבן עם text-shadow
- **כותרת:** 2.8rem, font-weight: 900
- **כפתורים:** גרדיאנט כחול-טורקיז + לבן שקוף
- **אנימציה:** staggerFadeIn עם delays

### 3. מוצרים חמים (Hot Products)
- **רקע כרטיס:** לבן עם מסגרת גרדיאנט
- **גודל:** 180px × 210px
- **אפקט 3D:** perspective: 1000px, translateZ
- **מרכז:** scale(1), opacity: 1
- **צדדים:** scale(0.85), opacity: 0.7
- **תג חיסכון:** גרדיאנט כחול-טורקיז, rotate(-8deg)

### 4. איך זה עובד (How It Works)
- **עיצוב:** Zigzag Timeline
- **קו מרכזי:** גרדיאנט אנכי כחול-טורקיז (3px)
- **אייקונים:** עגולים (76px), גרדיאנט, צל
- **טקסט:** כותרות #1e3a8a, תיאור #4b5563
- **Hover:** scale(1.15) על אייקון

### 5. טבלת מחירים (Price Table)
- **רשת:** 2×2 grid
- **כרטיסים:** לבן עם מסגרת גרדיאנט
- **מודגש:** רקע גרדיאנט מלא + תג "🔥 הכי משתלם!"
- **מחיר:** גרדיאנט טקסט (background-clip: text)
- **Hover:** translateY(-3px)

### 6. אין התחייבות (No Commitment)
- **קופסה:** לבן עם מסגרת גרדיאנט
- **אייקון:** עגול (50px), גרדיאנט
- **טקסט:** כותרת #1e3a8a, תיאור #4b5563

### 7. חבר מביא חבר (Referral)
- **כפתור פתיחה:** גרדיאנט, עגול (999px), צל חזק
- **פאנל:** לבן עם מסגרת גרדיאנט
- **קישור אישי:** input עם border #0891b2
- **מד התקדמות:** רקע #e0e7ff, מילוי גרדיאנט

### 8. קהל יעד (Target Audience)
- **רקע:** גרדיאנט מלא כחול-טורקיז
- **כרטיסים:** שקוף (rgba(255, 255, 255, 0.15)) + backdrop-filter
